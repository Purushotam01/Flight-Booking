# Flight Booking

A Flutter mobile application for searching, browsing, and booking flights. The app features an intuitive user interface for flight search, detailed flight information, booking results, and real-time connectivity monitoring.

## Features

- **Flight Search**: Search and filter available flights with ease
- **Flight Details**: View comprehensive information about selected flights
- **Search Results**: Browse and compare flight options
- **Connectivity Support**: Real-time connectivity status monitoring
- **Cross-Platform**: Supports Android, iOS, Web, and Desktop platforms
- **State Management**: Built with Provider for efficient state management
